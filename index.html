Hoi! Dit is de website van een kunstproject over de Stenen Tafel op Landgoed Mariendaal, dichtbij Arnhem, NL. Hiervoor verzamelen we veel beeldmateriaal, met als einddoel om een animatiefilm en/of boekje te maken, die uiteindelijk op locatie getoond zal worden. Op deze website zijn ingezonden foto’s te zien, genomen door voorbijgangers die op de oproep op locatie hebben gereageerd.

Hi! This is the website of an art project about the Stenen Tafel (Stone Table) at Landgoed Mariendaal, near Arnhem, the Netherlands. For this project we are collecting a lot of image material, with an end goal to create an animation video and/or book, which will eventually be shown on location. This website shows photos that were sent in by passers-by, who responded to the call on location.
